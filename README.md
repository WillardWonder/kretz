# Kretz AI Leadership and Governance Training - V2

This package contains the updated training materials aligned to the exact Kretz training outline and the governance-focused context from the Kretz/NTC planning emails.

## Files

- kretz_ai_leadership_governance_full_training_v2.pptx - full ready-to-deliver PowerPoint deck
- kretz_ai_leadership_governance_step_by_step_handout_v2.pdf - step-by-step participant/facilitator handout
- kretz_ai_leadership_governance_training_system_v2.html - updated web training page
- README.md - this file

## Suggested GitHub use

Upload the HTML, PPTX, and PDF files to the root of the existing WillardWonder/kretz repository. The HTML page links to the existing Kretz prototype files already in the repository:

- kretz-haul-planner.html
- kretz-landowner-prospecting.html
- Claude_Parcel.html
- claudeparcel.html

## Training focus

The updated materials emphasize:

1. Current AI tools and what they do well/not well
2. Real business impacts and AI News Break cautionary examples
3. Kretz-specific use cases and prototype demonstrations
4. Governance, risk, human review, approved tools, and policy
5. Kretz DNA, Sign-Off Rule, No-Fault Reporting, and Tool Builder role
6. Next steps for responsible adoption and ongoing adaptation
