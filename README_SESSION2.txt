KRETZ AI TRAINING — SESSION 2 FINAL SYNCHRONIZED PACKAGE
=========================================================
Repo: willardwonder.github.io/kretz

PURPOSE
-------
This package updates the Kretz training portal after Session 1 and adds the
final synchronized Session 2: Prompting in Practice materials.

UPLOAD
------
1. Extract this ZIP once.
2. Drag every extracted file and folder into the ROOT of the existing kretz
   GitHub repository.
3. Choose Replace for files that already exist.
4. Commit and push.
5. Refresh the GitHub Pages site.

Do not upload the ZIP itself into the repository. Do not place the extracted
files inside another subfolder.

BEHAVIOR RULES USED IN THE TRAINING
-----------------------------------
- AI fills in what you leave blank.
- AI is not a calculator.
- AI cannot forecast lumber prices.
- Never paste sensitive company information into an unapproved AI tool.
- If a number cannot be traced to a source, do not use it.
- A person signs off.

The facilitator notes provide the technical nuance underneath these memorable
employee rules without weakening the desired behavior.
